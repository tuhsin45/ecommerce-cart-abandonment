About Cart Abandonment Analysis

Project Overview

This comprehensive analysis examines cart abandonment patterns in the Brazilian E-Commerce dataset from Olist. The project provides actionable insights to help businesses reduce cart abandonment and increase revenue conversion.

Analysis Features

Data Processing
  • 99,000+ order records analyzed
  • Multi-table data integration
  • Real-time cart status classification
  • Product category mapping
Key Metrics
  • Overall abandonment rate calculation
  • Revenue impact assessment
  • Category-wise performance analysis
  • Geographic pattern identification

Technical Stack

Data Processing
  • Python 3.8+
  • Pandas for data manipulation
  • NumPy for calculations
  • SQL for database operations
Visualization
  • Plotly for interactive charts
  • Matplotlib for static plots
  • Seaborn for statistical graphics
  • Flask for web dashboard
Analysis Methods
  • Descriptive statistics
  • Cohort analysis
  • Geographic segmentation
  • Payment method analysis

Business Value

This analysis provides clear, actionable insights that can help businesses:

  • Reduce Cart Abandonment: Identify and address the primary causes of cart abandonment
  • Increase Revenue: Recover lost sales through targeted interventions
  • Optimize User Experience: Improve checkout processes based on data-driven insights
  • Strategic Planning: Make informed decisions about product portfolio and market focus

Data Sources

The analysis is based on the Olist Brazilian E-Commerce Public Dataset, which includes:

  • Order information (99,441 orders)
  • Customer demographics
  • Product categories and details
  • Payment transaction data
  • Order fulfillment tracking
  • Geographic location data
  • Seller information
  • Review and rating data

How to Use This Dashboard

  1. Dashboard: View real-time metrics and interactive charts
  2. API Endpoints: Access raw data programmatically
  3. Reports: Download detailed analysis results
  4. Refresh: Data updates automatically every 5 minutes
API Endpoints Available:
  • /api/summary - Overall statistics
  • /api/categories - Category analysis
  • /api/payments - Payment method analysis
  • /api/states - Geographic analysis
  • /api/charts/* - Chart data
